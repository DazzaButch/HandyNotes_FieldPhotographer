# HandyNotes: Field Photographer (Midnight Edition)

An updated, high-performance, and **taint-safe** version of the classic Field Photographer addon, specifically optimized for **World of Warcraft: Midnight (12.x)**.

## Features
- **Midnight Compatible:** Fixed the "Secret Number" crash and UI errors introduced in Patch 12.0.1.
- **Taint-Safe Logic:** Uses protected calls (`pcall`) for achievement queries to ensure stability in combat and instances.
- **Accurate Locations:** Shows exactly where to take S.E.L.F.I.E. camera photos for the *Field Photographer* achievement.
- **Smart Tracking:** Icons automatically disappear once you complete a specific location's criteria.
- **TomTom Integration:** Right-click any icon to set a waypoint (requires TomTom).

## Requirements
- [HandyNotes](https://www.curseforge.com/wow/addons/handynotes) (Must be installed)

## Installation
1. Download the latest **Release** (the `.zip` file, not the source code) from the [Releases page](https://github.com/DazzaButch/HandyNotes_FieldPhotographer/releases).
2. Extract the `HandyNotes_FieldPhotographer` folder into your `World of Warcraft/_retail_/Interface/AddOns/` directory.
3. Restart the game or type `/reload`.

## Configuration
This addon is a plugin for HandyNotes. You can find the settings here:
1. Open the game **Options** (Esc).
2. Navigate to the **AddOns** tab.
3. Select **HandyNotes** -> **Plugins** -> **Field Photographer**.

From there, you can adjust:
- **Icon Scale:** Make the camera icons larger or smaller.
- **Alpha:** Adjust how see-through the icons are.
- **Show on Continents:** Toggle whether icons show on the world map or just zone maps.
